Think you're a movie buff? Put your knowledge to the test with this interactive Movie Trivia game! This project challenges players to identify actors based on their filmography, combining entertainment with a test of cinema knowledge.

## How It Works

Players are presented with a list of movies and must correctly guess which actor has appeared in all of them. With each round, you'll discover just how well you know your favorite stars and their body of work.

## Technical Features

- Built with React and Next.js for a smooth, responsive user experience
- Integrates with TMDB (The Movie Database) API for accurate, up-to-date movie data
- Implements dynamic scoring system and progress tracking
- Features a clean, intuitive interface for seamless gameplay

## The Development Process

Creating this game involved careful consideration of user experience, data management, and performance optimization. The challenge was to create an engaging experience while handling complex API interactions and state management efficiently.
